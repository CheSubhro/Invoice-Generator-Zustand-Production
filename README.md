
# 🏬 B2B Commercial Invoice Generator System

An advanced, high-performance, responsive multi-step wizard invoicing application tailored for small to medium businesses (B2B/B2C). Built with state-of-the-art frontend technology, the engine provides lightning-fast arithmetic computations, zero data-loss states on tab crashes, and a classic pixel-perfect, **Tally-compliant paper/PDF printable tax invoice layout**.

## 🚀 Core Features

* **3-Step Interactive Wizard Dashboard:** Minimalist workflow breakdown separating profile configurations, core computations, and preview generation to minimize cognitive loading.
* **Dynamic Inventory Ledger System:** Add, update, or remove unlimited multi-product line items with instant cross-calculating field properties.
* **Automated GST Tax Engine:** Real-time extraction and compilation of internal metrics supporting custom input flags for **CGST**, **SGST**, and flat **Discount Cash Offs**.
* **Tally-Compliant Print System:** Native CSS `@media print` structural optimization flags to isolate operational headers completely from rendering onto final A4 hardcopies or PDF saves.
* **Chakra UI v3 Layout Engine:** Completely redesigned responsive grid stack pipelines preventing visual breakage on lightweight storage and billing handheld mobile devices.
* **Zustand Automated Core Processing:** Fast transient state tracking avoiding multi-render structural re-flows standard in old Context pipelines.

## 🛠️ Architecture Workflow Layout

The implementation breaks logical dependencies up sequentially into standalone wizard modules:

├── 🏢 Step 1: Business Profile & Customer Details (Store registration & buyer credentials)
│     └── ➔ Automatically Saves local system state parameters
├── 📦 Step 2: Product / Service Line Items (Ledger configuration, CGST/SGST/Discount rates)
│     └── ➔ Real-time calculation matrix evaluations
└── 📜 Step 3: Tally Invoice Sheet Blueprint (Printable preview compilation)
└── ➔ Triggers platform hardcopy pipelines, hidden navigation toolbars

## 💻 Tech Stack Specification

* **Core UI Primitive Framework:** React (Vite environment configuration)
* **Layout Structure & Design Token Engine:** Chakra UI v3 (Optimized style tokens)
* **Global State Processing Infrastructure:** Zustand State Management
* **Utility System Architecture:** Tailwind Utility Hooks / Flex configurations
